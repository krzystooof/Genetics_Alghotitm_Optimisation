start_button_action(controller, gui, checkboxes_one_set)

source code 

Function handling algorithm starting

Disable gui buttons for working time, connect with board, get configuration from gui entries and send it to board with starting command. Additional save selected parameter for further displaying (listboxes and graphs)

Parameters:
  • controller - used to communicate with board
  • gui - used to display results
  • checkboxes_one_set - rows with gui checkboxes that at least one option must be selected
